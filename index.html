JS FONT PARSER

A high-quality browser typography engine for JavaScript: parse fonts, shape text, inspect tables, and render glyphs with real outline access.

Open Demos Open Tools
Demos Animation-heavy showcases for outlines, particles, color fonts, and visual experiments. Tools Inspection surfaces for coverage, kerning, metadata, GSUB/GPOS, tables, and shaping checks. CLI Use the command-line explorer for metadata, language coverage, subsetting, SVG export, and stats.

Run from a local web server, then open http://localhost:8080/index.html.
