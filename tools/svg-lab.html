SVG Lab

Two different SVG workflows: generated outlines from glyph points, and embedded SVG glyphs from a font's SVG table.

Outline Export

Build SVG paths from parsed glyph outlines (works without an SVG table).

SVG Table Viewer

Render embedded SVG glyph documents from the font's SVG table.
If table payloads are compressed, browser support for DecompressionStream is required.
